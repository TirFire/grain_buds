# 🌿 小满日记 (GrainBuds) - V1.3.0

**小满日记** 是一款专注于私人情感记录、AI 智能辅助与极致数据安全的跨平台日记应用。在 V1.3.0 版本中，我们引入了深度的 AI 写作魔法与极速局域网快传功能，让记录变得更智能，让数据流动更自由。

## 🚀 下载与安装

请前往 GitHub Release 页面下载适用于您设备的最新安装包：

👉 **[点击此处前往下载最新的 Release 安装包](https://www.google.com/search?q=https://github.com/%E6%82%A8%E7%9A%84%E7%94%A8%E6%88%B7%E5%90%8D/%E6%82%A8%E7%9A%84%E4%BB%93%E5%BA%93%E5%90%8D/releases/latest)**

### 安装指引

  * **Android (.apk)**：下载后直接在手机端安装。如遇“来源不明”提示，请在设置中允许安装。
  * **Windows (.exe / .zip)**：下载安装包后运行或解压。
      * **注意**：首次运行开启局域网共享时，请务必在 Windows 防火墙弹窗中勾选“允许访问”权限，否则可能导致手机无法连接。

-----

## ⚙️ 快速配置

1.  **AI 配置**：进入“设置 -\> AI 配置”，填入您的 API Key 和 Base URL（如使用硅基流动或 DeepSeek）以激活魔法助手。
2.  **云同步**：在“设置”中配置 WebDAV 信息，实现数据的云端备份。
3.  **局域网同步**：两台设备连接同一 Wi-Fi，一端开启“发送模式”，另一端输入显示的 IP 地址即可。

-----

## 🔒 隐私与安全

  * **AES-256 加密**：所有本地存储与同步文件均支持高级别加密。
  * **本地优先**：数据存储在您的私人沙盒或您的 WebDAV 网盘中，开发者无法访问您的内容。
  * **AI 隐私**：AI 功能在本地解密后发送，建议不要对包含敏感账号密码的日记使用 AI 功能。
