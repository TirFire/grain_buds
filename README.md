### ✨ 为什么选择 GrainBuds？

* **🔒 数据主权归还用户**：所有日记内容、图片、音视频均 100% 离线保存在本地。我们不设服务器，不上传任何数据，您的隐私只属于您。
* **🛡️ 银行级加密**：支持单篇日记 AES-256-GCM 独立加密。即使数据库文件丢失，他人也绝对无法破解您的私密记录。
* **🎬 全能多媒体驱动**：集成工业级 `media_kit` 引擎。完美支持视频、Live 图、语音、文件附件混排，且在列表页支持静音循环预览。
* **🎨 沉浸式审美设计**：取消原生系统标题栏，采用自研沉浸式无边框 UI。内置 8 款精美主题色，支持深色模式与纸张护眼模式。
* **⏳ 跨越时空的记忆**：支持“历史上的今天”回顾，以及专为模糊记忆设计的“岁月深处（日期不可考）”功能。

### 📦 v1.1.0 更新日志 (What's New)

本次更新带来了重构级的体验提升：

* **【修复】** 彻底解决了 Windows 桌面端多媒体渲染导致的线程死锁与卡顿问题。
* **【优化】** 编辑页采用专业分栏布局，标题与工具栏固定，写作区域支持自动留白，体感更加丝滑。
* **【新增】** 回收站功能大升级：支持自动清理设置（7-90天）、多选批量恢复与彻底粉碎。
* **【新增】** 写作统计报表：包含 GitHub 风格打卡热力图、字数统计及连胜激励。
* **【新增】** 归档与收藏：支持集中查看已归档的只读日记与星标日记。

### ⚠️ 已知框架局限 (Known Issues)

由于 Flutter Windows 官方底层引擎对多行文本坐标上报的 Issue（#100083），在长文输入时，系统中文输入法（IME）的候选框可能会固定在左上角。我们已在 v1.1.0 中通过代码最大限度缓解了此现象，并持续关注官方修复进度。
